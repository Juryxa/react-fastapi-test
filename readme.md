# Как запустить приложение

0. Установите Python и Node.js на компьютер
1. Для запуска фронтенда выполнить команды по очереди в первом терминале
   - `cd client` - переход в папку client
   - `npm install` - устанавливаем зависимости
   - `npm run dev` - запускаем приложение
2. Для запуска сервера в другом терминале выполнить команды по очереди
   - `cd server` - переход в папку server
   - `pip install -r requirements.txt` - устанавливаем зависимости
   - `cd ..` - возвращаемся в корневую папку проекта
   - `uvicorn server.main:app --reload` - запускаем сервер
3. Переходим в браузер на адрес http://localhost:5173/
4. Готово
